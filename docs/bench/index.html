Path verdict

The cascade (webcodecs-worker → offscreen-worker → canvas-main → server-fallback) picks the best available path automatically. To force a specific path, the bench harness patches ImageDecoder and Worker via page.evaluate before calling compress().

  • ⚡ webcodecs-worker — best when available (Chrome 94+, Edge 94+, Safari 16.4+, Firefox 130+). GPU-accelerated decode.
  • 🥈 offscreen-worker — fallback for browsers without WebCodecs. OffscreenCanvas, ~10% slower.
  • 🥉 canvas-main — universal fallback (works everywhere, blocks main thread).
  • server-fallback — last-resort passthrough.
